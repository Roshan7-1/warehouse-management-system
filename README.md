# Enterprise Warehouse Management System (WMS)

## Overview
A cloud-based Warehouse Management System developed using Java Spring Boot and PostgreSQL.

## Features
- Product Management
- Inventory Tracking
- Warehouse Operations
- REST APIs
- PostgreSQL Integration
- Spring Security
- JWT Authentication (Upcoming)

## Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- Spring Data JPA
- Maven

## Modules Completed
- Spring Boot Setup
- PostgreSQL Integration
- Product CRUD APIs
- Service Layer
- Repository Layer
